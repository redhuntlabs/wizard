# Changelog

## [1.0.0] — 2026-04-21

Initial public release.

- Build AI spells for what you do.
- 25 bundled spells covering research, writing, decisions, planning, status updates, and dev workflows.
- 12 framework skills implementing the meta-builder, validator, inference engine, and adversarial tests.
- Slash commands: `/build-spell`, `/cast-spell`, `/list-spells`, `/refine-spell`, `/share-spell`, `/capture-this-chat`.
- Multi-tool support: Claude Code, Cursor, Codex, Gemini, OpenCode, Copilot CLI.
- Build-from-evidence: turn a working chat session, or any saved transcript, directly into a draft spell — no interview required. Try-it (Iron Law) still runs.
