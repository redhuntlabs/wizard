# Shape: Personalized Outbound Message

## Trigger keywords

cold email, outreach, intro, DM, reach out, message, pitch, cold message, intro email, network

## Output type

A message to a specific person with a specific ask.

## Strawman

```
Inputs:
  - Recipient name + role + company
  - One specific reason you picked this person (signal of relevance)
  - The single ask
  - Optional: prior interaction or shared connection

Steps:
  1. Open with the relevance signal in the subject line (one specific thing)
  2. First line: name the relevance signal again, briefly
  3. Body: 2-3 sentences max, leading to the ask
  4. Ask: one sentence, specific, time-boxed
  5. Sign off; no apology, no padding

Quality bar:
  - Under 100 words
  - Subject line names something specific to the recipient
  - The ask is one sentence and concrete (date / link / reply-by)
  - Zero "I hope this finds you well" type filler
```

## Common variations

- **Investor outreach**: ask is "15 min to discuss X" with a calendar link
- **Hiring outreach**: ask is "open to a 20-min chat about X role"
- **Network ping**: ask is "intro to person Y" with a one-line forwardable
- **Press / podcast**: ask is "quote on X by date Y" or "be a guest"
