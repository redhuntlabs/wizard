## Title
**Wizard v1.0.0 — Build AI spells for what you do**

## Body

Wizard is a meta-framework for turning any repeatable task into a reusable workflow — a spell — that any AI assistant can cast on your behalf. It works for engineers and non-engineers alike.

This is the first public release.

### What's in the box

- **25 bundled spells** covering research, writing, decisions, planning, status updates, code review, debugging, and more.
- **12 framework skills** implementing the meta-builder, validator, inference engine, kind-routing, and adversarial Iron-Law tests.
- **Slash commands**:
  - `/build-spell` — interview-driven meta-builder; routes to the right specialist by kind.
  - `/cast-spell` — run a specific spell on the current task.
  - `/list-spells` — browse the spellbook.
  - `/refine-spell` — update an existing spell (mandatory re-test).
  - `/share-spell` — export to vanilla `SKILL.md` format.
  - `/capture-this-chat` — turn the current chat session into a draft spell.
- **Build-from-evidence**: hand `/build-spell` a working chat or any saved transcript with `--from-transcript <path>` and Wizard will infer a strawman spell directly. The Iron Law (Stage 3 try-it) still runs — you can't ship a spell that hasn't been used end-to-end at least once.
- **Multi-tool support**: Claude Code, Cursor, Codex, Gemini, OpenCode, Copilot CLI. One spellbook, every assistant.

### Install

```bash
# Claude Code
/plugin install wizard@redhuntlabs

# Cursor — add the plugin from .cursor-plugin/plugin.json
# Or clone and run the installer for any supported tool:
git clone https://github.com/redhuntlabs/wizard.git
cd wizard && ./install.sh
```

Spells live at `$WIZARD_HOME` (defaults to `~/.wizard/`).
